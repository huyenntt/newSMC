checks if two nodes are parallel to each other

Checks if two Nodes are parallel to each other.

According to STA90: If there exist two distinct nodes n1 (this) and n2 (precondition 1) with identical preset (precondition 2) and identical postset (precondition 3) and the arc weight for each node of the pre- and postset to/from p1 equals the arc weiht to/from n2 appropriate (precondition 4) and, if both nodes are transitions, if they are connected to the same interface labels with the same weight (precondition 5), then these nodes are parallel to each other.

This function is needed by PetriNet::reduce_rule_3p() and PetriNet::reduce_rule_3t().

Parameters
n2the other node to which parallelism will be checked.
