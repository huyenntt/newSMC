Petri Net API: structural reduction.

This file is divided in 3 sections: Section 1 contains rules to remove unnecessary nodes, i.e. unused status places, transitions without preset or postset, dead nodes or initial marked places in choreographies. Section 2 contains rules from "Peter H. Starke - Analyse von Petri-Netz-Modellen" ([Sta90]). These rules preserve lifeness and boundedness, but no k-boundedness. Section 3 contains rules from "Thomas Pillat - Gegenüberstellung struktureller Reduktionstechniken für Petrinetze" ([Pil08]). These rules preserve lifeness and k-boundedness.
