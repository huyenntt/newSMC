compose constructor

Given two interfaces, ports with common labels will be composed and the result will be stored in this interface. Ports without a partner will be copied.

When composing a set of ports there must not remain any unused label (otehrwise an exception will be thrown).

By composition the given references label2label, label2place and commonLabels will be written.

netunderlying petri net
