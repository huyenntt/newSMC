libpnapi_is_present
char libpnapi_is_present ( )

Auxiliary function to check whether the PetriNet Net API is correctly installed.

This auxiliary function has the sole purpose to allow other tools to check whether they can link agains this PNAPI library using Autoconf's command "AC_CHECK_LIB(pnapi, libpnapi_is_present)".

Returns
Always 0 to indicate successful termination.
