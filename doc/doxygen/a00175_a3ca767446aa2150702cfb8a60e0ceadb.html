PNAPI_FOREACH
#define PNAPI_FOREACH (   I,
  C 
)    for(__typeof((C).begin()) I=((C).begin()); I != (C).end(); ++I)

a shortcut for STL iterations

Assume you have the code:

for (std::set<std::string>::iterator it = mySet.begin(); it != mySet.end(); ++it) {
// stuff
}

Then this code can be replaced by the following code

FOREACH(it, mySet) {
// stuff
}
Parameters
Ian iterator variable
Can container to iterate
Note
The code for the FOREACH macro was taken from a post on Stack Overflow http://stackoverflow.com/questions/197375/visual-c-for-each-portability/197926#197926
The type of the iterator is derived from the begin() function of the passed container C. There is no need to care about constness.
