Place & pnapi::Place::merge ( Place &  place,
bool  addArcWeights = true 
)

merges two places

Merges the properties of the given node into this one.

The following properties are merged:

  • NameHistory (concatenation)
  • WasInterface (disjunction)
  • Capacity (maximum)
  • TokenCount (maximum)
  • Pre-/Postset (union)
Precondition
the places must reside in the same PetriNet instance
Postcondition
the place is internal after merging
